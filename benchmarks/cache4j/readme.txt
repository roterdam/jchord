
 cache4j - кеш для java объектов

 Возможности
 -----------

   - кеширование объектов в памяти
   - спроектирован для работы в многопоточных приложениях
   - две реализации кеша synchronized, blocking
   - реализовано алгоритмы вытеснения объектов LFU, LRU, FIFO
   - для хранения объектов могут использоваться сильные (strong) или магкие(soft) ссылки
   - распостраняется под BSD лицензией

 Требования к Java
 -----------------
   cache4j можно использоваться с JDK 1.3, 1.4, 1.5
   При использовании с JDK 1.3 нужен xml парсер (например: xerces)

 Установка
 ---------
   Для работы с cache4j нужен cache4j_?.jar + файл конфигурации

 Документация
 ------------
   Для получения более подробной информации откройте doc\tutorial.htm
   Документация в формате javadoc doc\api\index.html

 Планируется
 -----------
   - вытеснение устаревших объектов на диск
   - работа в кластере
   - тестирование и реализация других алгоритмов вытеснения LFU2, LRU2, ...
